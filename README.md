# node-cache-clear
a tool for clearing dns & sockets pool & htsts cache.

## Install 

~~~
npm i node-cache-clear -g
~~~

## Usage 

**Add domains**
~~~
iclear add test.com test.com
~~~

**Remove domain**
~~~
iclear remove a.com 
iclear clear // remove all domains
~~~

**Show domain configs**
~~~
iclear show
~~~

**Run**
~~~
iclear // run the cache clear script
~~~
